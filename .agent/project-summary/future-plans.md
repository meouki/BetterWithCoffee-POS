# Future Plans & V2 Roadmap

This document outlines features that are planned or partially mocked up but not yet fully implemented or integrated into the main PulsePoint POS lifecycle.

## Receipt System
- **ReceiptModal**: A post-checkout receipt display showing store info, order ID, timestamp, itemized list with modifiers, subtotal/VAT/total, payment method, tendered amount, change, and actions to "Print Receipt" or start a "New Order". Currently existing in the codebase but inactive in the main flow.
