# Better with Coffee — POS System

A production-grade, modular Point of Sale (POS) system designed for specialty café operations. Built with a focus on speed, tactile feedback, and premium aesthetics.

## 🚀 Current Features

### 1. The Cashier's Instrument (POS Interface)
- **Touch-First Design**: Optimized for mobile and tablet devices with high-tap targets.
- **Dynamic Menu**: Categorized product grid with "Fly-to-Cart" animations.
- **Modifier System**: Advanced bottom sheets for sugar levels, milk alternatives, and add-ons.
- **Order Flow**: Required Dine-In/Take-Out selection and seamless checkout.

### 2. The Manager's Cockpit (Admin Dashboard)
- **KPI Overview**: Visualized data for today's revenue, order counts, and best-sellers.
- **Role-Based Access Control (RBAC)**:
  - **Master**: Full system control (Users, Products, Inventory).
  - **Admin**: Read-only oversight, reports, and settings.
  - **Cashier**: Focused POS terminal access.
- **Menu Management**: Full CRUD operations for products with availability toggles.
- **User Management**: Add, edit, and manage staff accounts with password security.

### 3. Personal Profile & Performance
- **User Identity**: Custom avatar and role identification.
- **Daily Time Record (DTR)**: Integrated calendar grid for tracking attendance.
- **Personal Stats**: Real-time tracking of revenue and orders processed by the logged-in user.

### 4. Design System
- **Preset Aesthetic**: "Night Shift" (Dark Premium) theme.
- **Responsive**: Fully fluid layout from 360px phones to 4K desktops.
- **Tech Stack**: React + Vite, Vanilla CSS with CSS Modules, Lucide Icons.
